Pasos en Git

Por AlumnoUDA

tips de pasos para usar git

>>git init

>>> git config --global user.email "tu correo electronico"

>>> git config --global user.name " tu nombre"

>>> git status

>>> git add "nombre archivo"

>>> git commit -m "escribir comentario"

>>> git diff : para ver los cambios del documento

>>> git status

>>> git commit --m "comentario de confirmación"

>>> git log --online : ver los comentarios

>>> git branch : muestras las ramas creadas

>>> git branch: " nombre de rama" crea una rama con el nombre

>>> git merge " rama que desa conviar" : merge siempre se ejecuta en la rama master

>>> git resert "43ses2w" --soft :se devuelve al punto anterio sin borrar el trabajo posterior
